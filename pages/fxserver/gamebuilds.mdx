# Game Builds

## What Are they?

The `Game Build`, is the version of the game you are running, often defined as the `DLC version`.
Using this allows you to unlock and use certain parts of `GTA 5`/ `Red Dead Redemption`. These parts include, but are not limited to:
- Vehicles
- Weapons
- Peds
- Objects
- Clothing
- Animations
- Interiors (MLOs)

## Setting Game Build

To set the game build your server uses, find the number that matches the DLC you want, from the lists below.

You will Then want to navigate to your <DefinitionText text={"server.cfg"}/> (or use the <DefinitionText text={"CFG Editor"}/>) and insert the following:

```cfg
sv_enforceGameBuild <number>
```
> Note: Insert without the `<>`
